Project Overview

TETRIS

Autonomous AI-Driven Trading Engine

Overview

TETRIS is a fully autonomous AI-driven algorithmic trading platform engineered specifically for the highly volatile Indian stock market (NSE). Designed to remove human emotion from trading, it acts as a relentless, cold logic engine.

Key Highlights

  • Trades 50 Nifty stocks simultaneously
  • Multi-timeframe analysis matrix
  • Ensemble machine learning models
  • Rule-based strategy fallbacks
  • Institutional-grade risk management constraints

Dockerized Microservice Architecture

Built for resilience and uptime. The platform runs on 4 independent services managed via Docker Compose, exposing 12+ FastAPI REST endpoints for real-time data ingestion and model execution.

  • Real-time signal generation engine
  • Automated trade execution layer
  • Instant Telegram alerts pipeline
  • Automated PDF performance reporting

Command Dashboard Features

  • Real-time P&L monitoring
  • Live signal dashboard
  • Market regime detection
  • Portfolio analytics & Risk metrics

Project Overview

FRIDAY

Privacy-First Local AI Assistant

The Principle

No cloud dependency. No API keys. No subscriptions. Everything runs locally on the user’s machine. FRIDAY is an autonomous AI assistant inspired by Tony Stark’s systems, giving you absolute data sovereignty without sacrificing intelligence.

Core Architecture

  • Whisper: Real-time neural speech recognition
  • Mistral-7B: Local LLM for advanced reasoning
  • Coqui TTS: High-fidelity voice synthesis
  • ChromaDB: Persistent vector memory
  • React: Real-time visual dashboard interface

Capabilities

More than just a chatbot, FRIDAY is an active agent capable of reaching into your digital life to organize, retrieve, and execute tasks based on natural voice conversations.

  • Natural voice conversations
  • Local LLM spatial reasoning
  • Long-term memory retention
  • Calendar management & scheduling
  • Slack workflow automation
  • Local file organization
  • Daily situational briefings
  • Complex task orchestration
